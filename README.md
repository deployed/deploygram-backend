## Description
API for the deploygram - our simplified version of the instagram that will be the basis for workshops with students.

## Installation

```bash
$ yarn

```

## Running the app

```bash
# Postgres database
$ docker-compose up -d

# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
